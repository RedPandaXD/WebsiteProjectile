Automatic

The automatic type of projectile usually holds at least 15 rubber bands, and can hold as much as 40. They could possibly hold more, but then the performance would be slightly worse. These usually have very fast reload spitting out 20 rubber bands in 5 seconds. The drawback, however, is that these usually have poor range and aim. These could be used to just spray the target, which guarantees a hit either way. More accurate ones could be made, but these could never beat the semiautomatic or bolt actions rubber band projectiles in accuracy. On the bright side, these are very powerful at close range.

Challenges

The main obstacle I found while building automatic projectiles is the fact that it has to repeatedly fire very fast as well as function very smoothly, so it would not jam and destroy the satisfaction. Since I use rubber bands as the projectile being shot, the most common way to overcome this problem is to take a large cylinder, hook the rubber bands along the length of the cylinder, then load it over string, so when the string is pulled, the cylinder spins and causes the rubber bands to fire. the cylinder and string have to be very durable, withstanding a lot of pressure. To make it automatic, I then hooked up a motor strong enough to pull the string on the side, and all that was left was making the projectile aimable. Automatic projectiles do not have to be very accurate, but it would not be well built if it only shot to one side. Aligning the rubber band's midair motion is more or less a guesswork, going through several trials and errors to finally come to a satisfactory result.
